# Alert

Alert presents feedback or important information to users.

```html
<div class="Alert Alert--success" role="alert">
  <div class="Alert__icon"></div>
  <div class="Alert__message">Message</div>
</div>
<div class="Alert Alert--danger" role="alert">
  <div class="Alert__icon"></div>
  <div class="Alert__message">Message</div>
</div>
```
